> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add route-based edit-and-resubmit flows for rejected resubmittable expense and leave requests without manually editing generated Kubb client code.

**Architecture:** Keep the implementation inside the existing page structure: list/detail/create/edit page components in `src/pages/index.tsx`, with thin TanStack Router file routes under `src/routes/_private`. Reuse the current create-form markup and option arrays directly for the two edit pages, prefilling fields from `findOne` responses and submitting through the generated resubmit mutation hooks.

**Tech Stack:** React 19, TypeScript, TanStack Router, TanStack Query generated by Kubb, Vitest/Testing Library, NestJS/Jest backend service tests.
